#  Todo App

A simple Todo Application built using **HTML, CSS, and JavaScript**.  
It helps users add, manage, and delete daily tasks easily.

---

##  Features
-  Add new tasks
-  Delete tasks
-  Mark tasks as completed (if implemented)
-  Simple and clean UI
-  Fast and lightweight

---

##  Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

---

##  How to Run
1. Clone or download the project
2. Open `index.html` in browser
3. Start adding tasks
